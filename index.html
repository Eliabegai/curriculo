Backend Biblioteca de Leitura Digital

No backend da Biblioteca de Leitura Digital, foi utilizado NestJS para criar uma API estruturada e escalável, com Prisma como ORM para gerenciar o banco de dados. A persistência dos dados foi configurada com Docker, permitindo uma instância de banco de dados local, garantindo desenvolvimento e testes mais ágeis e isolados. A API oferece endpoints seguros para autenticação de usuários, cadastro e consulta de históricos de leitura, mantendo os dados sincronizados com o frontend e prontos para consultas rápidas.
